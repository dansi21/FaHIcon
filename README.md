# F@H SVG Badge

'F@H SVG Badge' is a small tool I put together for people to be able to proudly display their 'Folding@home' stats on their GitHub profile or any other location icons are commonly found.

Please feel free to leave any recommendations in either issues or emailed to me directly.

# How To Add To Your Profile

To add one of the SVGs to your profile it is as easy as pasting the following into your github or readme.

## Total Points

```[![{Your Username} F@H Stats](https://fah.dansi.dev/icon/points?userId={Your Username})](https://stats.foldingathome.org/donor/name/{Your Username})```

[![Dansi21 F@H Stats](https://fah.dansi.dev/icon/points?userId=dansi21)](https://stats.foldingathome.org/donor/name/dansi21)

## Leaderboard Rank

```[![{Your Username} F@H Stats](https://fah.dansi.dev/icon/leaderboard?userId={Your Username})](https://stats.foldingathome.org/donor/name/{Your Username})```

[![Dansi21 F@H Stats](https://fah.dansi.dev/icon/leaderboard?userId=dansi21)](https://stats.foldingathome.org/donor/name/dansi21)
